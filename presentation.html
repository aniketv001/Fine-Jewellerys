14 ‚Äî Challenges

Challenges & Solutions

Challenge

MongoDB Atlas DNS SRV records couldn't be resolved by local ISP DNS server, causing backend connection failure

‚Üí
Solution

Replaced mongodb+srv:// URI with direct shard hostnames discovered via Google DNS (8.8.8.8), bypassing SRV lookup entirely

Challenge

Wishlist adding silently failed ‚Äî async API calls were fired without await, so UI state never updated from server response

‚Üí
Solution

Added optimistic UI updates, proper async/await in all handlers, boolean return values, and state revert on API failure

Challenge

Rollup native module @rollup/rollup-darwin-arm64 missing due to npm optional dependency bug on Apple Silicon Mac

‚Üí
Solution

Deleted node_modules and package-lock.json, then ran a clean npm install to correctly resolve platform-specific binaries
